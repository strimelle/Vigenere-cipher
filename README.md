# Vigenere Cipher

A Java Maven project for implementing a Vigenère cipher encryption and decryption system with a graphical user interface.

## Project Description

This application allows the user to:

- enter plaintext
- enter an encryption key
- encrypt text using the Vigenère cipher
- decrypt ciphertext back to the original plaintext

This project is being developed as a practical assignment in the field of information security.

## Technologies Used

- Java 21
- Swing
- Maven

## Features

The application includes the following functionality:

- Vigenère encryption
- Vigenère decryption
- user input validation
- graphical user interface
- basic mode, where only alphabetic characters are encrypted
- extended ASCII mode

## Project Structure

Main classes:

- `Main` – application entry point
- `VigenereUI` – graphical user interface
- `VigenereCipher` – encryption and decryption logic
- `InputValidator` – input validation

## How to Run

1. Open the project in IntelliJ IDEA.
2. Make sure Java 21 is selected.
3. Run the `Main` class.

## Project Goal

The goal of this project is to create a working Vigenère cipher encryption and decryption system and apply the classical cipher algorithm in practice.
